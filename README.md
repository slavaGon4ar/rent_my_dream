# Rent My Dream

Rent My Dream - это back-end приложение для системы аренды жилья, включающее управление объявлениями, поиск, фильтрацию и бронирование.

## Технологии

- Django & Django REST Framework
- MySQL
- Docker
- Swagger (drf-yasg)
